## 可修改变量
| 环境变量    |      默认值      |      建议      |
| ----------- | :--------------: | :------------: |
| VERSION     |      1.10.2      | *强烈建议升级* |
| HOME        |      /home       |     可修改     |
| PORT        |       3000       |     可修改     |
| ADMIN_EMAIL |  test@test.com   |    建议修改    |
| DB_SERVER   | mongo(127.0.0.1) |   不建议修改   |
| DB_NAME     |       yapi       |   不建议修改   |
| DB_PORT     |      27017       |   不建议修改   |
| VENDORS     | ${HOME}/vendors  |   不建议修改   |
